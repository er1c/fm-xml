## Documentation

Links:

- [Website](https://er1c.github.io/scala-woodstox/)
- [API documentation](https://er1c.github.io/scala-woodstox/api/)


### Usage

Add this to your `build.sbt`:

```scala
libraryDependencies += "io.github.er1c" %% "scala-woodstox" % "1.0.0-RC1"
```
